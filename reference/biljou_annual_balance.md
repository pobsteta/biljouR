# Annual water balance summary

Sums the elementary water fluxes by calendar year so that \\P \approx
In + T + Eu + D + \Delta W\\ can be checked.

## Usage

``` r
biljou_annual_balance(run)
```

## Arguments

- run:

  A
  [`biljou_run`](https://pobsteta.github.io/biljouR/reference/biljou_run.md)
  object.

## Value

A data frame of yearly flux totals (mm).
